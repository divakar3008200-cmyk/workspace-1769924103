How to run

1. Open index.html in your browser (double-click or use Live Server in VS Code).

Features

- 8x8 board (64 squares)
- Snakes and ladders drawn with SVG lines/curves
- Two players, roll dice, animated moves
- Responsive layout and attractive UI with gradients and shadows
- Easy to customize snakes/ladders mappings in script.js

Customization

- To change board size: adjust SIZE in script.js and update CSS square size calculations.
- To change snakes/ladders: edit the "jumps" object in script.js using start->end mappings.

Notes

This is a simple, client-side implementation meant for learning and small games. Feel free to extend: add multiplayer over network, sound, AI, or more players.
